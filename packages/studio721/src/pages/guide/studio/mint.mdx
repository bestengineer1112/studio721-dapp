Studio 721 Mint is a free tool for configuring an NFT minting interface that you
can share with collectors.

## Features

- 🤯<SpacerHorizontal inline size={20} />Create a custom NFT minting interface
  without writing code
- 🖋<SpacerHorizontal inline size={20} />Works with most ERC-721 contracts (not
  just Studio 721 contracts)
- ⚙️<SpacerHorizontal inline size={20} />Configurable UI design
- 🏠<SpacerHorizontal inline size={20} />Your config is stored in your browser
  url (no centralized servers)
- 🔓<SpacerHorizontal inline size={20} />Free and open source!
