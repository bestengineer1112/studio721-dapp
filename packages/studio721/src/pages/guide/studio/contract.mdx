Studio 721 Contract is a free tool for configuring, compiling, deploying, and
verifying custom [ERC-721](https://eips.ethereum.org/EIPS/eip-721) NFT smart
contracts.

## Features

- 🤯<SpacerHorizontal inline size={20} />Create a custom NFT contract without
  writing code
- 🚀<SpacerHorizontal inline size={20} />Compile, deploy, and verify all without
  leaving your browser!
- ⚙️<SpacerHorizontal inline size={20} />Lots of configuration options
- 🏠<SpacerHorizontal inline size={20} />Your config is stored in your browser
  url (no centralized servers)
- 💸<SpacerHorizontal inline size={20} />Gas-optimized deployment to save you
  money
- 💻<SpacerHorizontal inline size={20} />Royalties (ERC-2981) and multiple
  payout addresses
- 👩‍💻<SpacerHorizontal inline size={20} />You can still download and customize
  the code if you want
- 🔓<SpacerHorizontal inline size={20} />Free and open source!
